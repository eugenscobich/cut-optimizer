🔨 Panel Cutting Optimizer

Welcome to the Panel Cutting Optimizer! This web application helps you generate optimized cutting patterns for sheet materials while minimizing waste.

ℹ️ Quick Start: Click the button below to launch the application
Launch Application →

Features

📊 CSV Import

Import parts and stocks from CSV files for quick data entry

🎯 Smart Optimization

Recursive algorithm finds all valid cutting solutions

📈 Analytics

Detailed statistics on waste, cuts, and material usage

🎨 Visual Rendering

Interactive canvas with zoom and pan controls

💾 Auto-Save

Browser local storage preserves your data automatically

⚡ Performance

Asynchronous optimization with stop capability

Sample Data

The application comes with sample CSV files for testing:

  • sample-parts.csv - Example parts list
  • sample-stocks.csv - Example stock sheets
💡 Tip: Use the sample data to explore the application features before using your own data.

System Requirements

  • Modern web browser (Chrome, Firefox, Safari, Edge)
  • JavaScript enabled
  • Local storage support (for auto-save)

Help

After launching the application:

  1. Add Parts: Manually or upload CSV
  2. Add Stocks: Manually or upload CSV
  3. Configure Settings: Set kerf thickness and cut margins
  4. Run Optimization: Click "Start Optimization"
  5. Review Results: Select solutions to view details and visualizations

For detailed documentation: See README.md in the project folder
